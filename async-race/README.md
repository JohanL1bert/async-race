# async-race

#### How to work with this

1. Go to https://github.com/mikhama/async-race-api and download server
2. Install all dependencies that are described on the server page
3. Start server
4. Go to deploy page https://johanl1bert-async-race.netlify.app/
5. Testing

##### Additional info:

For comfortable testing you can use Postman
Server start on http://127.0.0.1:3000
